# 🧠 Online Quiz System (Java + MySQL)

## 📌 Project Overview

The **Online Quiz System** is a desktop-based application developed using **Java (Swing)** and **MySQL**. It allows users to log in, take timed quizzes, and view their scores. The system stores user data, quiz questions, and results in a database.



## 🎯 Objective

To build a **timed quiz application** with:

* User authentication
* Randomized questions
* Timer-based quiz
* Score calculation
* Result storage and display



## 🛠️ Technologies Used

* **Java (Swing)** – GUI development
* **JDBC** – Database connectivity
* **MySQL** – Database
* **IDE** – VS Code / IntelliJ / Eclipse



## 🗄️ Database Schema

The system uses the following tables:

1. **users** – Stores login credentials
2. **quizzes** – Quiz details
3. **questions** – Quiz questions and options
4. **results** – Stores user scores



## ⚙️ Features

✔ User Login System
✔ Random Question Loading
✔ Timer-Based Quiz
✔ Automatic Score Calculation
✔ Result Storage in Database
✔ Result Display Screen
